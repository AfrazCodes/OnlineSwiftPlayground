SwiftPlayground.run

Online Swift Playground. Build with Swift, for Swift

Finally! Self hosted and embeddable solution with a third-party modules support baked in.

Features

  • Simplicity - Build and run with Swift Package Manager.
  • macOS & Linux - Run on macOS or/and Linux.
  • Custom module - Import custom frameworks, that makes is perfect tool to play with your code.
  • Swift stack - Swift on Server stack.
  • Hosted - Easy to deploy own instance with Docker.
  • Open Source1 - The open source code, nothing to hide. (available soon)
